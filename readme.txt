Yet Another Contact Form.

== Description ==

mtx contact form using jQuery to implement the form in a modal window.

 
== Installation ==


1. Upload folder mtx-contact-form to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Proceed to the configuration in the settings menu. You can use the short tag [blogname] and [name]
1. mtx contact form is activated by any link with class mtx_contact_form_open
